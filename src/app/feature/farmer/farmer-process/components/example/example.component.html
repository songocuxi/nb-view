Sử dụng Angular Material https://material.angular.io/, góc trên bên phải trang web có để phiên bản, nhớ chọn lại v16

1. Vào trang Angular Material, chọn Component, chọn component muốn sử dụng, bấm quá tab 'API', copy dòng import ❴ ... ❵ from " @angular/… "
2. Dán dòng import đó vào module cha của component muốn sử dụng material (module chứa component đó), ví dụ muốn sử dụng metarial trong component process-detail, thì import vào process-management.module.ts. Khai báo tên module trong phần @NgModel(❴import❵)
3. Quay lai trang Angular Material, qua tab Overview , chọn biểu tượng “< >”, copy code html để vào file html của component đó (nếu đoạn html đó có sử dụng biến, hàm thì bấm qua TS xem và copy hoặc khai báo tương ứng lại)

Sử dụng Font Awesome:

1. Chọn icon, để ý tên class của icon, ví dụ < i class=" fa-solid fa-circle-user " >< / >
2. Tại component muốn sử dụng icon, trong file .ts import ❴ faCircleUser ❵ from '@fortawesome/free-solid-svg-icons', nhấn từ từ hệ thống sẽ đề xuất.
3. Trong phần export class, khai báo biến với giá trị là icom name, ví dụ: tempAvatar (biến này đặt tên gì cũng được) = faCircleUser (đây là cái đã import bên trên), xem file example.component.ts
4. Trong file .html, sử dụng thẻ

Sử dụng các thành phần trong file scss-config, đặt trong thư mục src/assets: @import "path đến file scss-config"
